# 🌍 Sistema de Banco de Dados de Países

Projeto de modelagem e implementação de um banco de dados relacional normalizado para armazenamento de informações de países.

## 📌 Descrição
O sistema foi desenvolvido para organizar dados geográficos e econômicos de países, incluindo informações como continente, moeda, idioma, população e PIB.

## ⚙️ Tecnologias Utilizadas
- SQL
- MySQL

## 🚀 Funcionalidades
- Modelagem de banco de dados relacional normalizado
- Implementação de relacionamentos (1:N e N:N)
- Consultas com JOIN, GROUP BY e agregações
- Criação de VIEW para visualização estruturada
- Gerenciamento de usuários e permissões
